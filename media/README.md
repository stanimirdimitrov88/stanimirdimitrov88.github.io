# Adding Project Images

This directory is for your project screenshots and visualizations.

## Image Files Needed

Add your images with these exact filenames to display them on your project pages:

1. **twitter-placeholder.jpg** - Twitter sentiment analysis visualization
2. **dashboard-placeholder.jpg** - Work injuries dashboard interface
3. **xray-placeholder.jpg** - Chest X-ray classification results
4. **wardwatch-placeholder.jpg** - WardWatch deployment tool
5. **coral-placeholder.jpg** - Coral detection visualization
6. **podia-placeholder.jpg** - PODIA project overview

## Image Guidelines

- **Format:** JPG or PNG
- **Recommended width:** 1200-1600px (will scale responsively)
- **Aspect ratio:** 16:9 or similar landscape format works best
- **File size:** Keep under 500KB for fast loading

## How to Add Images

1. Save your images in this `media/` folder
2. Name them exactly as listed above (or update the `src` paths in the HTML files)
3. The images will automatically appear on the respective project pages

## Alternative: Update Filenames

If you prefer different filenames, update the `<img src="media/YOUR-FILENAME.jpg">` paths in:
- project-twitter.html
- project-dashboard.html
- project-xray.html
- project-wardwatch.html
- project-coral.html
- project-podia.html
