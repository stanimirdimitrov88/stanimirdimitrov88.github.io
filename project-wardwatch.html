WardWatch - Burglary Prediction in London

Interactive dashboard for burglary risk assessment

Created an interactive dashboard for burglary risk in London to assist with police deployment. Performed feature engineering and utilized an XGBoost model to predict expected burglaries in different boroughs of London. Developed interactive dashboard visualizing burglary risk across London boroughs with comprehensive feature engineering to extract predictive signals. Implemented XGBoost machine learning model for burglary prediction and created data-driven tool to optimize police resource deployment. Provided actionable insights for law enforcement strategic planning.
