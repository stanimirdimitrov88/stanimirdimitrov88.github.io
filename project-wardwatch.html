WardWatch – burglary forecasting

Forecasting weekly burglaries for London LSOAs using XGBoost and LSTM, with SHAP explainability and baselines.

Overview

WardWatch predicts short-term burglary counts for each Lower Super Output Area (LSOA) in London. I engineered temporal features and exogenous signals (seasonality, holidays, weather proxies) and compared classical baselines (naive, seasonal naive, moving averages) to modern ML models.

Python Pandas XGBoost PyTorch LSTM Time series SHAP
GitHub Repository

Results

  • ML models beat naive and seasonal naive baselines across most LSOAs.
  • XGBoost delivers best average sMAPE improvement; LSTM competitive for highly seasonal segments.
  • SHAP indicates strong weekly seasonality and holiday proximity effects on predictions.
  • Error clustering reveals geographic pockets with higher volatility benefitting from richer features.

Stack & Approach

  • Feature pipeline: rolling stats (7 / 28 day), lag features, holiday flags, month/week encoding.
  • XGBoost with time-based CV splits and early stopping on validation sMAPE.
  • LSTM sequence-to-one architecture (window 28 → next week) with dropout & gradient clipping.
  • Explainability: SHAP summary + per-LSOA force plots for local insight.
  • Evaluation: sMAPE + MASE + per-segment ranking; error analysis via residual distribution.
