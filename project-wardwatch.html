WardWatch – burglary forecasting

Predictive policing / spatial–temporal forecasting for London LSOAs.

Overview

WardWatch is a predictive model that forecasts burglary risk for London LSOAs. It compares XGBoost, LSTM and simple statistical baselines, and explains feature importance via SHAP.

Tech stack

  • Python (pandas, scikit-learn, xgboost)
  • LSTM (Keras / TensorFlow or PyTorch)
  • SHAP for explainability
