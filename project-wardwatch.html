WardWatch - Burglary Prediction in London

Interactive dashboard for burglary risk assessment to assist with police deployment.

Overview

Created an interactive dashboard for burglary risk in London to assist with police deployment. Performed feature engineering and utilized an XGBoost model to predict expected burglaries in different boroughs of London.

Python XGBoost Feature Engineering Interactive Dashboard Predictive Modeling

Key Achievements

  • Developed interactive dashboard visualizing burglary risk across London boroughs.
  • Performed comprehensive feature engineering to extract predictive signals from crime data.
  • Implemented XGBoost machine learning model for burglary prediction.
  • Created data-driven tool to optimize police resource deployment.
  • Provided actionable insights for law enforcement strategic planning.

Technical Stack

  • Machine Learning: XGBoost for burglary prediction modeling
  • Feature Engineering: Temporal, spatial, and contextual features
  • Visualization: Interactive dashboard for risk assessment
  • Geographic Analysis: Borough-level predictions across London
  • Application: Police deployment optimization
