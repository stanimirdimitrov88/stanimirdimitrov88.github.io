2D Coral Area Estimation

Computer vision for marine ecology monitoring.

Overview

Utilized an EPFL model to detect and estimate coral coverage in 2D underwater imagery. This project supports marine ecology research and coral reef health monitoring.

Python Computer Vision EPFL Model Marine Ecology

Technical Approach

  • Applied EPFL model for automated coral detection in underwater images.
  • Implemented 2D area estimation algorithms for coral coverage quantification.
  • Processed and analyzed underwater imagery datasets.
  • Contributed to marine ecology monitoring and conservation efforts.

Applications

  • Automated monitoring of coral reef health over time.
  • Quantitative assessment of coral coverage in marine ecosystems.
  • Support for conservation and restoration initiatives.
  • Scalable analysis of large underwater image datasets.
