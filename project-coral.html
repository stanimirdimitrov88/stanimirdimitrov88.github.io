2D Coral Area Estimation

Automated computer vision pipeline for marine ecology

Developed an automated computer vision pipeline that estimates 2D coral reef area from seabed images using object detection, pixel-to-cm ratio calculation, and coral segmentation using EPFL-ECEO model. Pipeline includes automated detection of coral structures in underwater imagery, calibration system for accurate area measurement, and precise segmentation using EPFL-ECEO model. Enables automated monitoring of coral reef health and coverage over time with scalable analysis of large underwater imagery datasets. Supports marine ecology research, conservation efforts, and quantitative assessment for reef restoration projects.
