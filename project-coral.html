2D Coral Area Estimation

Automated computer vision pipeline for marine ecology

Developed an automated computer vision pipeline that estimates 2D coral reef area from seabed images using object detection, pixel-to-cm ratio calculation, and coral segmentation using EPFL-ECEO model.

Technical Pipeline

  • Object Detection: Automated detection of coral structures in underwater imagery
  • Pixel-to-CM Ratio Calculation: Calibration system for accurate area measurement
  • Coral Segmentation: Precise segmentation using EPFL-ECEO model
  • Area Estimation: Automated 2D coral coverage quantification
  • Pipeline Automation: End-to-end processing of seabed image datasets

Applications & Impact

  • Automated monitoring of coral reef health and coverage over time
  • Scalable analysis of large underwater imagery datasets
  • Support for marine ecology research and conservation efforts
  • Quantitative assessment tool for reef restoration projects
  • Reduced manual annotation time for marine biologists
