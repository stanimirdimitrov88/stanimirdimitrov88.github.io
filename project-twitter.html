Twitter Data Sentiment Analysis

Role: Scrum Master

Performed NLP analysis on 40GB of Twitter data to compare customer support performance of major airlines in the global market. Identified specific topics requiring improvement based on sentiment trends.

Key Achievements

  • Analyzed 40GB of Twitter data comparing customer support across major airlines
  • Utilized RoBERTa model for sentiment analysis to gauge customer satisfaction
  • Implemented LDA (Latent Dirichlet Allocation) for topic detection
  • Created MongoDB database to store customer-support conversations
  • Identified specific improvement areas based on sentiment trends

Technical Stack

  • NLP Framework: RoBERTa for sentiment classification
  • Topic Modeling: LDA for topic detection and clustering
  • Database: MongoDB for conversation storage
  • Data Volume: 40GB Twitter dataset
