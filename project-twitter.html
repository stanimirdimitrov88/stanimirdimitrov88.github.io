Twitter Data Sentiment Analysis Project

NLP analysis on 40GB of Twitter data comparing customer support performance across major airlines.

Role: Scrum Master

This project analyzed 40GB of Twitter data to compare customer support performance of major airlines in the global market. I identified specific topics requiring improvement based on sentiment trends.

Python NLP RoBERTa LDA MongoDB

Key Achievements

  • Performed NLP analysis on 40GB of Twitter data to compare customer support performance across major airlines.
  • Utilized a RoBERTa model for sentiment analysis to gauge customer satisfaction.
  • Implemented LDA (Latent Dirichlet Allocation) for topic detection to identify recurring issues.
  • Created a MongoDB database to store and manage conversations between customers and customer support teams.
  • Identified specific topics that require improvement based on sentiment trends and topic clustering.

Technical Stack

  • NLP Framework: RoBERTa for sentiment classification
  • Topic Modeling: LDA for topic detection and clustering
  • Database: MongoDB for conversation storage
  • Data Volume: 40GB Twitter dataset
  • Analysis Focus: Airline customer support comparison
