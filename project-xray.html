Classification of Chest X-ray Images

Deep learning for thoracic disease detection with ethical analysis.

Overview

Fine-tuned Deep Convolutional Neural Networks for classification of thoracic diseases from chest X-ray scans. Implemented data augmentation techniques and utilized two pre-trained models: DenseNet121 and ResNet50.

Python Deep Learning DenseNet121 ResNet50 Medical Imaging

Technical Approach

  • Fine-tuned pre-trained DenseNet121 and ResNet50 architectures for thoracic disease classification.
  • Implemented comprehensive data augmentation techniques to improve model generalization.
  • Conducted thorough performance evaluation comparing both model architectures.
  • Applied transfer learning strategies to leverage pre-trained ImageNet weights.

Ethical Considerations

  • Shortcut Bias: Analyzed potential for models to exploit dataset artifacts rather than learning true disease patterns.
  • Cognitive Bias: Evaluated how model predictions might influence clinical decision-making.
  • Uncertainty Analysis: Implemented methods to quantify prediction confidence and identify edge cases.
  • Documented limitations and appropriate use cases for clinical deployment.
