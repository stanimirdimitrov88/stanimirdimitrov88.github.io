Chest X-ray Classification

Deep learning for thoracic disease detection

Fine-tuned Deep Convolutional Neural Networks for classification of thoracic diseases from chest X-ray scans. Implemented data augmentation techniques and utilized two pre-trained models: DenseNet121 and ResNet50.

Technical Approach

  • Fine-tuned pre-trained DenseNet121 and ResNet50 architectures
  • Implemented comprehensive data augmentation techniques
  • Conducted thorough performance evaluation comparing both models
  • Applied transfer learning strategies leveraging ImageNet weights

Ethical Considerations

  • Shortcut Bias: Analyzed potential for models to exploit dataset artifacts
  • Cognitive Bias: Evaluated how predictions might influence clinical decisions
  • Uncertainty Analysis: Quantified prediction confidence and edge cases
  • Documented limitations and appropriate clinical use cases
